### 2. Cara Upload File ke GitHub
Karena kamu sudah punya repository di GitHub, ikuti langkah ini untuk memasukkan semua file PHP kamu:

1.  Buka repository **Uasweb.** di browser.
2.  Klik tombol **"Add file"** lalu pilih **"Upload files"**.
3.  Buka folder `C:\xampp\htdocs\sembakobarokah` di komputermu.
4.  **Drag & Drop** (tarik) semua file (`index.php`, `models.php`, folder `config`, dll) ke kotak upload di GitHub.
5.  Tunggu proses selesai, lalu ketik pesan commit: **"Final version Sembako Barokah"**.
6.  Klik tombol hijau **"Commit changes"**.
